
# Geometry Calculator

This project was made for calculating the measurements of many shapes. 

## How to Use the Project

1. Input the shape you want to create
2. Input the the measurements
3. Look at the shapes

## Key Features

- Generates measurements for
  - squares
  - triangles
  - circles
  - rectangles
  - cubes
  - spheres
  - cones
  - pyramids
  - prisms
