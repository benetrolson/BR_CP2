
# Character Creator

This project is an enhanced RPG Character Manager that allows users to create, visualize, and analyze characters.

## How to Use the Project

1. Input the character you want to create
2. Look at their info
3. Create more characters

## Key Features

- Keeps track of characters
    - Data visualization
    - Statistical analysis
    - Random background/description maker
    - Comparison tools
- Allows you to make new ones
- Random quest giver

## Installation Instructions

1. Install faker
2. Install matplotlib
3. Install pandas
4. Install numpy

## How to Run

Run the program with:
- python main.py
