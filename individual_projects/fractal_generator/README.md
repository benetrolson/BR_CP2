# Fractal Generator

This project was made for generating fractals like the sierpinski triangle and the koch snowflake. By taking the recursion depth, color, background color, and type of fractal, it will generate the desired shape. So far, this can only generate the sierpinski triangle and the koch snowflake. 

## How to Use the Project

1. Input the recursion depth
2. Input the color
3. Input the background color
4. Input the type of fractal
5. Pull up the Python Turtle Graphics
6. Look at the fractal
7. Rinse and repeat

## Key Features

- Generates fractals

## Instalation Instructions

1. Install Python
2. Install Turtle
3. Run the code
